API Primitives and Basic Objects

  • API resources: objects that represent the state of the cluster
  • Pods: smallest deployable units in Kubernetes
  • ReplicaSets: ensures a specified number of replicas of a pod are running
  • Deployments: declarative updates for Pods and ReplicaSets
  • Services: abstraction that defines a logical set of Pods and a policy to access them
